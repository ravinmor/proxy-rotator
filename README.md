# Proxy rotator


## Installing

### Package manager

Using npm:

```bash
$ npm install proxy-rotator
```
Using yarn:

```bash
$ yarn add proxy-rotator
```

Once the package is installed, you can import the library:
```js
import Protator from 'proxy-rotator';
```
